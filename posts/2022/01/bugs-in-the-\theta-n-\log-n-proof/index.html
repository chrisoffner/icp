ü™≥ Bugs in the $\Theta (n \log n)$ proof

The proof starts off by defining a round $j$:

Assume that the points of $A$, except for the leftmost one, are assigned to $b_{n‚àíj+1}$ and $b_{n‚àíj+2}$, for some $1 \le j \le n$ (clearly, these assignments can involve only two consecutive Voronoi cells), and consider all iterations of the algorithm, in which some points of $A$ cross the common Voronoi boundary $\beta_{n‚àíj+1}$ of the cells $\mathcal{V}(b_{n‚àíj+1}), \mathcal{V}(b_{n‚àíj+2})$. We call the sequence of these iterations round $j$ of the algorithm.

‚ö°Ô∏è For $j=1$ the point $b_{n-j+2} = b_{n+1}$ does not exist, so instead of the mentioned $1 \le j \le n$ it should read $2 \le j \le n$.

The proof claims that in each iteration of round $j$ except its last one, exactly $j$ points cross boundary $b(j)$:

$(ii)$ at each iteration in this round, other than the last one, the overall number of points of $A$ that cross $\beta_{n‚àíj +1}$ is exactly $j$, and no point crosses any other boundary

‚ö°Ô∏è Counterexample 1: ![[counterexample1.svg]] For $n = 5, j = 2$ we can see that in iteration 4, $\beta_3$ is crossed by $a_2$, making iteration 4 the last iteration of round $3$. At the same time, $\beta_4$ is also crossed by $a_n$, making iteration 4 also the first iteration of round $2$. In this first iteration of round $2$ we have only one point cross $\beta_4$, which contradicts both claims that in each but the last iteration of a round, exactly $j$ points ($j=2$ here) would cross its boundary, and that no other boundary would be crossed (the previous boundary $\beta_3$ is also crossed).

‚ö°Ô∏è Counterexample 2: $n = 6, j = 3$ ![[counterexample2.svg]] The same situation can be observed whenever the last iteration of a round is also the first iteration of the next round, i.e. whenever two boundaries are crossed in the same iteration.

The proof claims that in the last iteration of round $j$, the number $l$ of points in $A'$ to the left of $b(j)$ is less than $j$: ($0 \le l < j)$:

Thus, at each iteration of round $j$ (except for the last one), in which there are points of $A$ that remain in the cell $\mathcal{V}(b_{n‚àíj+1})$, the $j$ rightmost points of $A$ (among those contained in $\mathcal{V}(b+{n‚àíj+1}))$ cross $\beta_{n‚àíj+1}$. Let us now consider the last such iteration. In this case, all the points of $A$, except $l$ of them, for some $0 \le l < j$ (and the leftmost point, which we ignore), have crossed $\beta_{n‚àíj+1}$ in previous iterations.

‚ö°Ô∏è The problem with $\boldsymbol{0 \le l}$ How could $l = 0$ in the last iteration of a round? ü§î If $l = 0$, i.e. there remain no points in $\mathcal{V}(b_{n‚àíj+1})$, then, by the definition of a round, we cannot be in the last iteration of round $j$ anymore.

‚ö°Ô∏è The problem with $\boldsymbol{l < j}$ ![[counterexample3.svg]] For $n = 7, j = 3$ we see that iteration 7 is the last iteration of round $3$, i.e. in this iteration $\beta_5$ is crossed for the last time. Contrary to the claim $l < j$, we have $l = j = 3$ here.

The proof claims that in the last iteration of round $j$ exactly $j-1$ points cross any boundary:

$(iii)$ at the last iteration of the round, the overall number of points of $A$ that cross either $\beta_{n‚àíj+1}$ or $\beta_{n‚àíj +2}$ is exactly $j ‚àí 1$.

![[counterexample3.svg]] Using the same counterexample as above, for $n = 7, j = 3$ we can see that in the last iteration of round $3$, exactly $3 = j$ points (not $j - 1$!) cross $\beta_5 = \beta_{n‚àíj+1}$.
